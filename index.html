Output:



Type in a primary color with a semi colin to change the color of the output text. You may also type "rainbow" for an alternating rainbow color.
The input box contains an HTML code compiler. You may use inline CSS in a text tag to change the output text to a custom color.

Script Explanation:

Using HTML, we create an input box and an output section. Using Javascrpt events, we collect the user input. That input is then assigned to a variable (valinput). Next, we use "let x = document.getElementById("id");" to get the output section. The value of the variable (valinput) is assigned to that ID. That will create the output text.
